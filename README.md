# Personal Portfolio

My personal portfolio website built with Next.js and TypeScript.

## Live Demo
[portfolio-blue-phi-50.vercel.app](https://portfolio-blue-phi-50.vercel.app)

## Tech Stack
- **Framework**: Next.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI**: shadcn/ui
- **Package Manager**: pnpm
- **Deployment**: Vercel

## Getting Started
```bash
git clone https://github.com/mrsandeep27/portfolio.git
cd portfolio
pnpm install
pnpm dev
```
